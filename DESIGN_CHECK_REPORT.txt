Interface 2046 v4 design check

Fixed:
1. Right-side route label
   - Graduation Research page now shows "Undergraduate Research" instead of the full project title.
   - Other pages use short semantic route labels.
   - Overlong fallback route names are truncated.

2. Sticky section labels
   - Section labels no longer stick in the viewport.
   - They scroll naturally with the page.

3. Light Mode ordinary text links
   - Home / Contact ordinary links now use a darker blue (#0a4f7a).
   - Underline remains clean; no glass box is reintroduced.

4. Home Research / CV CTAs
   - Replaced generic pill buttons with 2046-style trajectory command tabs.
   - Research is the primary launch command.
   - CV is a secondary dossier command.
   - Mobile size and touch ergonomics are preserved.

Remaining design proposals:
1. The right-side vertical route rail is useful on desktop, but it can be hidden on medium-width laptops if it feels too close to content.
2. The Solar System diagram is now a strong identity element; avoid adding more HUD elements around it.
3. Paper Shelf can later receive a denser archive layout once paper count grows.
4. Photography could later use a film-contact-sheet index below the slideshow, but only when the page becomes photo-heavy.
