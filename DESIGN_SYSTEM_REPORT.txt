Unique Site System v1

Implemented:
1. Design-system consolidation
   - Defines stable visual tiers:
     Tier 1: major project/card surfaces
     Tier 2: status/internal fields
     Tier 3: tags/chips/badges
   - Adds consistent colour variables for Space Mode and Light Mode.
   - Keeps the current Interface 2046 direction but makes components easier to maintain.

2. Mission Log scientific record structure
   Each Mission Log entry now follows:
   - Context
   - Operation
   - Observation
   - Interpretation
   - Next Action
   - Figures

3. Practicality
   - Figure grids remain compatible with the existing lightbox.
   - Research Log / Graduation status auto-sync is preserved.
   - Existing data-log-* attributes are preserved.
   - Mobile layout becomes one-column for the record fields.

Future proposals:
1. Paper Shelf can be converted into a reading-dossier structure:
   Role in research / Key idea / Method relevance / Status.
2. Photography can later receive a contact-sheet index once the photo count grows.
3. If CSS grows too large, the next major step should be file-level refactoring:
   base.css, components.css, pages.css, interface-2046.css.
